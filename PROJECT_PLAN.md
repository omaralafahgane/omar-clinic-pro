## 🏗️ Architecture

```
Frontend (Next.js)
    ↓
API Routes (Next.js)
    ↓
Supabase (PostgreSQL)
```

| Layer | Technology |
|-------|-----------|

| Backend | Next.js API Routes |
| Database | PostgreSQL (Supabase) |
