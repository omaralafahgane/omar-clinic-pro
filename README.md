# Omar Clinic Pro

نظام إدارة عيادة طبية متكامل وحديث يوفر حلاً شاملاً لإدارة العمليات الطبية والإدارية.

## 🎯 المميزات الرئيسية

- **إدارة المرضى**: تسجيل وتتبع بيانات المرضى الشاملة
- **إدارة المواعيد**: جدولة وإدارة مواعيد المرضى بكفاءة
- **إدارة الموظفين**: إدارة الأطباء والموظفين والجداول الزمنية
- **نظام الفواتير**: إنشاء وتتبع الفواتير والدفع
- **التقارير والإحصائيات**: تحليلات شاملة لأداء العيادة
- **الأمان**: نظام مصادقة آمن وتشفير البيانات
- **الواجهة الحديثة**: تصميم سهل الاستخدام وسريع الاستجابة

## 📁 هيكل المشروع

```
omar-clinic-pro/
├── frontend/              # تطبيق الويب (React + TypeScript)
├── backend/               # خادم API (Python/Node.js)
├── database/              # ملفات قاعدة البيانات
├── docs/                  # التوثيق
├── tests/                 # اختبارات المشروع
└── README.md             # هذا الملف
```

## 🚀 البدء السريع

### المتطلبات
- Node.js 16+ أو Python 3.8+
- npm أو yarn
- قاعدة بيانات (MySQL/PostgreSQL)

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/omaralafahgane/omar-clinic-pro.git
cd omar-clinic-pro

# تثبيت الاعتماديات
npm install
# أو
pip install -r requirements.txt
```

### التشغيل

```bash
# تشغيل الخادم
npm start
# أو
python app.py
```

الوصول إلى التطبيق على: `http://localhost:3000`
