# Omar Clinic Pro 🏥

نظام متكامل لإدارة العيادات الطبية (SaaS) مصمم باحترافية لتلبية احتياجات الأطباء والعيادات في الوطن العربي.

## 🚀 الميزات الرئيسية

- **إدارة المرضى:** سجلات طبية كاملة، تاريخ مرضي، وتتبع الحالة.
- **جدولة المواعيد:** نظام ذكي لإدارة المواعيد وتجنب التضارب.
- **النظام المالي:** إصدار فواتير احترافية وتتبع الدفعات.
- **نظام الاشتراكات:** إدارة اشتراكات العيادات عبر مفاتيح التفعيل والحماية بالميدل وير.
- **لوحة تحكم Super Admin:** تحكم كامل في العيادات والمستخدمين وتوليد مفاتيح التفعيل.
- **تكامل Resend:** إرسال تنبيهات البريد الإلكتروني الحقيقية (ترحيب، إعادة تعيين كلمة مرور، تنبيهات اشتراك).
- **الأمان:** حماية بيانات كل عيادة بشكل مستقل باستخدام Supabase Row Level Security (RLS).

## 🛠️ التقنيات المستخدمة

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Clerk
- **Email Service:** Resend
- **Deployment:** Vercel

## ⚙️ الإعداد والتشغيل

1. **تثبيت التبعيات:**
   ```bash
   pnpm install
   ```

2. **إعداد المتغيرات البيئية:**
   قم بإنشاء ملف `.env.local` وأضف المفاتيح الخاصة بـ Supabase و Clerk و Resend كما هو موضح في `SUPABASE_SETUP_INSTRUCTIONS.md`.

3. **تطبيق قاعدة البيانات:**
   تأكد من تشغيل ملفات الـ Migrations الموجودة في مجلد `database/migrations` بالترتيب الصحيح.

4. **تشغيل المشروع:**
   ```bash
   pnpm dev
   ```
